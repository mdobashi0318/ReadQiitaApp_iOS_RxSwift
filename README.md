# ReadQiitaApp_iOS_RxSwift


## setup
1. Clone the project.
```
   git clone https://github.com/mdobashi0318/ReadQiitaApp_iOS_RxSwift.git
   cd ReadQiitaApp_iOS_RxSwift
   xcodegen generate
```
or 
```
   download zip
   cd ReadQiitaApp_iOS_RxSwift-1.1.0
   xcodegen generate
```
2. Opne ReadQiitaApp_iOS_RxSwift.xcodeproj
